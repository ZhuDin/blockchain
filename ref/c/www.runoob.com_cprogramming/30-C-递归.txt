C 递归

递归指的是在函数的定义中使用函数自身的方法。

语法格式如下：

	void recursion()
	{
	   statements;
	   ... ... ...
	   recursion(); /* 函数调用自身 */
	   ... ... ...
	}
	 
	int main()
	{
	   recursion();
	}

C 语言支持递归，即一个函数可以调用其自身。但在使用递归时，程序员需要注意定义一个从函数退出的条件，否则会进入死循环。

递归函数在解决许多数学问题上起了至关重要的作用，比如计算一个数的阶乘、生成斐波那契数列，等等。
